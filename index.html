Quick Install

Get up and running in seconds with Cargo.

bash

# Install via cargo

cargo install flowsentinel

# Or build from source

git clone https://cuty.io/GithubLink

cd flowsentinel && cargo build --release

sudo ./target/release/flowsentinel
