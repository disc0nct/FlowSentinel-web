Capabilities

Deep System Visibility

Process-Centric

Aggregate traffic by PID, UID, and Name. See live Bytes/s and Packets/s metrics for every running application.

DPI Engine

Extract TLS SNI to see encrypted domains. Log HTTP Hosts and track full DNS query history for every process.

Forensics

Record PCAP files per process. Set bandwidth triggers to auto-record suspicious spikes. Log events to SQLite.

Container Aware

Automatically detects Docker and Kubernetes (CRI-O) containers. Maps traffic to specific container IDs.

Advanced Threat Detection

Threat Intelligence

Integrate custom IP blacklists. Malicious connections are highlighted in BOLD RED instantly.

SQLite Audit Logs

Persistently log every DNS query and connection event for post-incident audits and timeline reconstruction.

Process Lineage

View Parent PID (PPID) and full command lines to detect suspicious spawned processes.

Install via Cargo

Requires Linux (Kernel 4.0+), libpcap, and Rust MSRV 1.75+.

BASH

# Install dependencies

sudo apt install libpcap-dev libsqlite3-dev


# Clone & Build

git clone https://github.com/disc0nct/FlowSentinel

cd FlowSentinel

cargo build --release


# Run with privileges

sudo ./target/release/flowsentinel
